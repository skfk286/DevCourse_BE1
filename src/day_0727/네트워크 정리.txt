IP 주소
- 네트워크 상에서 특정 지점을 식별하기 위한 주소값
- InetAddress
PORT
- ip 주소로네트워크 식별 이후 해당 장치내에서 데이터를 송/수신 하기위한 통로. 숫자로 식별됨
- 프로그램이 외부와 통신하기 위해서는 특정 번호의 포트를 반드시 점유해야 함.

Socket
- 특정 포트에 데이터 송수신을 위해 꽂아두는 객체. (전원 플러그 같이 꼽아두는 애를 상상)

UDP
- 비연결성. 연결을 생성하는 시간이 들지 않음.
- TCP 대비 빠름. 실시간으로 빨리 전달해야 하는 경우 선택함.
- 비신뢰성. 패킷 유실 가능성이 있음
- 실시간 라이브 전송

TCP/IP
- 연결성. 연결을 생성하고 유지하는데 시간과 메로리 비용이 들어감.
- UDP 대비 느리다고 함.
- 신뢰성. 상대방이 현재 접속중임을 확인한 상태에서 데이터를 송/수신 함으로 무조건 도착한다는 보장이 있다.
- 우리가 실시간 방송처럼 패킷 유실 하지 않는 대부분의 네트워크 통신은 TCP 기반이다.
- ServerSocket, Socket
